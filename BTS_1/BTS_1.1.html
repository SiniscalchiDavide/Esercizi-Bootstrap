Esercizio BTS_1.1 – Bootstrap Get Started

1. A cosa serve Bootstrap?

Bootstrap è un framework front-end open source che aiuta a creare siti web responsive e mobile-first in modo semplice e veloce. Fornisce stili CSS predefiniti

2. Cosa devi importare per creare una pagina web con Bootstrap?

È necessario importare il file CSS e il file JS di Bootstrap, solitamente tramite CDN:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script>

3. Cos'è un CDN? Che vantaggi comporta rispetto a scaricare Bootstrap sul tuo sito?

Un CDN (Content Delivery Network) è una rete di server sparsi nel mondo che permette di fornire contenuti (come librerie CSS e JS) in modo veloce e affidabile.
Vantaggi:

  • Non devi salvare i file localmente
  • Caricamento più veloce grazie alla rete di distribuzione
  • Risparmi banda e spazio sul tuo server
  • I file sono spesso già in cache nel browser dell'utente

4. A cosa serve questo attributo del tag meta?

<content="width=device-width, initial-scale=1"> serve per rendere la pagina responsive.

Imposta la larghezza del viewport in base alla larghezza del dispositivo e la scala iniziale al 100%. È essenziale per una corretta visualizzazione sui dispositivi mobili.

5. Che differenza c'è tra Container e Container Fluid?

Container crea un contenitore con larghezza fissa che si adatta al breakpoint (responsive, ma centrato).
Mentre container-fluid crea un contenitore che occupa il 100% della larghezza dello schermo, indipendentemente dalle dimensioni del dispositivo.
